# リフレクションアプリ（仮）
## 概要
- 目的：主にパートナー間の不仲を解消する支援
  - 利用方法：いくつかの質問に答え、シェアする

## 今後の実装予定
- SNS(LINE)での共有機能
- SNSを利用したログイン機能
- ウィザード形式での入力
